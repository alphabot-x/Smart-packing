How It Works

1
Model the problem: Each item has a weight (kg) and a value score (necessity + comfort, 1–10).
2
Build DP table: Fill dp[i][w] = max value using first i items with capacity w.
3
Traceback: Walk back through the table to find which items were selected.
4
Result: The optimal subset that maximizes value within your bag's weight limit.
